========================================
  HarmonyOS规则爬虫工具 - GUI版本
  项目文件清单
========================================

📁 新增的GUI相关文件：

1. gui_app.py
   - 主GUI应用程序
   - 包含完整的图形界面和功能
   - 可直接运行：python gui_app.py

2. build_exe.spec
   - PyInstaller打包配置文件
   - 定义了打包参数和依赖

3. build.bat
   - 自动打包脚本
   - 一键打包成EXE程序
   - 双击即可运行

4. 启动GUI.bat
   - 快速启动GUI程序
   - 双击即可运行
   - 带错误提示

5. GUI使用说明.md
   - 详细的使用指南
   - 包含常见问题解答
   - 完整的操作流程

6. README_GUI.md
   - 项目快速入门文档
   - 功能特性介绍
   - 快速开始指南

7. .env
   - 配置文件（程序自动生成）
   - 存储API密钥等配置
   - 已添加到.gitignore

========================================
使用方式：
========================================

【方式一：直接运行GUI】
1. 双击"启动GUI.bat"
2. 或在命令行运行：python gui_app.py

【方式二：打包成EXE】
1. 双击"build.bat"
2. 等待打包完成
3. 在 dist\HarmonyOS规则爬虫工具\ 找到EXE文件

【方式三：手动打包】
1. pip install pyinstaller
2. python -m playwright install chromium
3. pyinstaller build_exe.spec --clean

========================================
快速开始：
========================================

1. 安装依赖：
   pip install -r Requirements.txt
   python -m playwright install chromium

2. 启动GUI：
   双击"启动GUI.bat"

3. 配置API：
   - 输入Gemini API密钥
   - 点击"保存配置"

4. 开始爬取：
   - 点击"开始爬取"
   - 查看实时日志

5. 获取结果：
   - 点击"打开输出目录"
   - 查看 final_cursor_rules 文件夹
